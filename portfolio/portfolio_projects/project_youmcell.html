Youm Cell External Inspection Machine

Flexible Cell 타겟의 외관 검사 시스템

1. 프로젝트 개요

반도체/전극 검사 공정에서 발생하는 미세 결함을 검출하기 위한 시스템 개발 프로젝트.

2. 문제 상황 (Before)

  • 미세 결함 검출 정확도 부족
  • 조명 변화에 따른 오검출 발생
  • Alignment 오차로 인한 검출 불안정

3. 해결 방법 (How)

  • Sub-pixel alignment 알고리즘 적용
  • 조명 정규화 및 이미지 전처리 개선
  • ROI 기반 검사 구조 설계

4. 핵심 기술

  • Sub-pixel Alignment 기반 위치 정렬
  • 영상 필터링 및 노이즈 제거
  • 딥러닝 기반 결함 분류 (CNN)

5. 결과 (After)

  • 검출 정확도 +15% 향상
  • 오검출율 30% 감소
  • 검사 속도 20% 개선

6. 트러블슈팅

  • 조명 변화로 인한 threshold 실패 → adaptive 방식으로 개선
  • Alignment drift 발생 → 기준점 보정 로직 추가
